# React Slide Puzzle

Forked from https://github.com/erinlynndownes/react-slide-puzzle

## Getting started

   ### start

    First clone the repo to your local server and then install dependencies and start.
    ```
    npm install
    npm start
    ```
   ### build
    npm run build
